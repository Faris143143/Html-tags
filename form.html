Form Tags

  • <form> - Represents an HTML form.
  • <input> - Represents an input control.
  • <label> - Represents a label for an input control.
  • <textarea> - Represents a multi-line text input control.
  • <select> - Represents a drop-down list of options.
  • <option> - Represents an option in a drop-down list.
  • <optgroup> - Groups related options within a drop-down list.
  • <fieldset> - Groups related form elements together.
  • <legend> - Represents a caption for a <fieldset>.
  • <button> - Represents a clickable button.
  • <datalist> - Provides a list of predefined options for an input control.
  • <output> - Represents the result of a calculation or user action.
  • <progress> - Represents the progress of a task.
  • <meter> - Represents a scalar measurement within a known range.
