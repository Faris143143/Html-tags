Meta Tags

  • <meta charset="character_set"> - Specifies the character encoding for the HTML document.
  • <meta name="viewport" content="..."> - Sets the viewport properties to control the layout and scaling of the web page on different devices.
  • <meta name="description" content="..."> - Provides a brief description of the web page's content.
  • <meta name="keywords" content="..."> - Specifies a list of keywords related to the web page's content.
  • <meta name="author" content="..."> - Specifies the author of the web page.
  • <meta http-equiv="refresh" content="seconds;url=URL"> - Redirects or refreshes the page after a specified number of seconds.
  • <meta name="robots" content="..."> - Controls how search engine robots crawl and index the web page.
  • <meta name="generator" content="..."> - Specifies the software or content management system used to create the web page.
  • <meta name="og:title" content="..."> - Provides the title of the web page for social media sharing.
  • <meta name="og:description" content="..."> - Provides a description of the web page for social media sharing.
  • <meta name="og:image" content="..."> - Specifies the image to be displayed when sharing the web page on social media.
  • <meta name="og:url" content="..."> - Specifies the URL of the web page for social media sharing.
