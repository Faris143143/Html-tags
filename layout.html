Layout tags

  • <header> - Represents the introductory content or a group of introductory content at the top of a document or section.
  • <nav> - Represents a section of a page that contains navigation links.
  • <main> - Represents the main content of a document. It should be unique and not nested within other structural elements.
  • <article> - Represents a self-contained composition in a document, such as a blog post, article, or news story.
  • <section> - Represents a standalone section of content that is semantically related.
  • <aside> - Represents a section of a page that contains content tangentially related to the main content, such as sidebars or pull quotes.
  • <footer> - Represents the footer of a document or a section. It typically contains information about the author, copyright information, or links to related documents.
  • <figure> - Represents self-contained content, such as images, diagrams, illustrations, or code examples, that is referenced in the main content but can be moved away from it.
  • <figcaption> - Represents a caption or legend for the content within a <figure> element.
  • <div> - A generic container element that does not carry any semantic meaning, but can be used for grouping and styling purposes.
