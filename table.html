Table Tags

  • <table> - Represents a table, which is a grid of rows and columns.
  • <caption> - Represents the caption or title of a table.
  • <thead> - Represents the header section of a table.
  • <tbody> - Represents the body section of a table.
  • <tfoot> - Represents the footer section of a table.
  • <tr> - Represents a row within a table.
  • <th> - Represents a header cell within a table.
  • <td> - Represents a data cell within a table.
  • <colgroup> - Specifies a group of one or more columns in a table for styling purposes.
  • <col> - Specifies attributes for one or more columns within a <colgroup> element.
