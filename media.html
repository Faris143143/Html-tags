Media Tags

  • <img> - Represents an image in the HTML document.
  • <audio> - Represents sound content, such as music or audio streams.
  • <video> - Represents video content, such as movies or video streams.
  • <source> - Specifies multiple media resources for the <video> and <audio> elements.
  • <track> - Specifies timed text tracks for the <video> and <audio> elements.
  • <canvas> - Provides a drawing space for creating graphics and animations using JavaScript.
  • <svg> - Defines scalable vector graphics for rendering shapes, images, and text.
  • <map> - Defines an image map for clickable regions within an <img> element.
  • <area> - Defines a clickable area within an image map.
  • <iframe> - Embeds an external web page or document within the current HTML document.
